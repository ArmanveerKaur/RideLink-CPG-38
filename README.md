# RideLink-CPG-38
Capstone Project 
To run the website you need to put following commands in the terminal:
1. cd desktop/rpi_scripts/
2. rm trip_log.csv
3. rm events_log.csv
4. python receiver2.py

Now you need to simultaneously open another instance of terminal:
1. cd desktop/rpi_scripts/
2. python website3.py
Now your website is live and you can open it at the designated address displayed on the terminal
This website is also accessible on another device on the same wifi network via host address displayed on terminal

 To demostrate the WiFi based passenger entry-exit log you will have to:
 1. Open the app
 2. Ensure that your WiFi is turned ON
 3. Now press the "Connect to WiFi button to connect to systema nd log you entry
 4. To exit you have to click the "Disconnect" button OR you can simply move out of WiFi range

 To demostrate the RFID based passenger entry-exit log you will have to:
 1. Tap the RFID card for entry
 2. To exit you need to tap the RFID card again
